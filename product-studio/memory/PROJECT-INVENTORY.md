## Usage Rules

- **CEO reads this file first** at every heartbeat before creating a new task or project
- Before opening any new brief, confirm: is this already listed above in active or paused state?
- When a project moves stages, update the status here (live → shipped, etc.)
- When the active section is empty, that is a success state — do not generate tasks to fill it
- Do not mirror project content here — use the linked task/issue for full detail; this file is an index only
