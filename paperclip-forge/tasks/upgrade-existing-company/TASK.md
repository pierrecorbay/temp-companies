Start an upgrade engagement for an existing company.

Expected flow:

1. Inventory the current package and identify drift, duplication, and missing governance.
2. Redesign workflow, files, and skill references where needed.
3. Patch the package with the minimum justified change set.
4. Audit and review the result before CEO signoff.
