# CEO Heartbeat

1. Classify the request: create, upgrade, or reject.
2. Confirm `PROJECT-INVENTORY.md` exists and has been updated for this engagement.
3. Dispatch intake to the Chief of Staff.
4. Review the proposed workflow, skill map, and implementation scope before build work continues.
5. Require a producer-side self-check from the Staff Engineer.
6. Require a structural audit from the Quality Auditor.
7. Require a blocking final review from the Code Reviewer.
8. Approve, return for correction, or stop.

If no useful work remains, stop. Idle is success.
